Anlisis y Visualizacin de Datos del Triatln Ironman

Este cuaderno explora los datos histricos del campeonato Ironman. A travs de visualizaciones y anlisis detallados, se estudian los tiempos, gneros y pases participantes para comprender mejor las dinmicas de este evento deportivo.

Descripcin de los Datos

Los datos contienen informacin sobre los resultados del Ironman con las siguientes columnas:

  • Year: Ao del evento.
  • Place: Posicin final del atleta.
  • Athlete: Nombre del atleta.
  • Country: Pas del atleta.
  • Time: Tiempo total registrado.
  • Gender: Gnero del atleta.

# Visualizacin 5: Densidad de Valores Promedios por Pas y Gnero

x_values = data_avg[data_avg['Gender'] == 'Male']['Time (hrs)']  # Valores de hombres
y_values = data_avg[data_avg['Gender'] == 'Female']['Time (hrs)']  # Valores de mujeres

# Crear el grfico de densidad con contornos
fig = ff.create_2d_density(
    x=x_values,
    y=y_values,
    colorscale='Greens',
    hist_color='rgba(100, 255, 120, 0.2)',
    point_size=2,
    title="Densidad de Valores Promedios: Hombres vs Mujeres"
)

# Mejorar la visualizacin
fig.update_layout(
    xaxis_title="Promedio de Valores (Hombres)",
    yaxis_title="Promedio de Valores (Mujeres)",
    template="plotly_white"
)
